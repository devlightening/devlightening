### 🚀 About & Core Competencies

I design and build end-to-end software systems — from mobile apps to distributed backends. Modern backend engineering is not merely about writing code; it is about architecting sustainable, flexible, and high-availability systems.

* **Ecosystem:** Developing enterprise-grade solutions within **.NET 8** and **Spring Boot** ecosystems.
* **Architecture:** Implementation of **Clean Architecture** and **Domain-Driven Design (DDD)** with strict layer separation.
* **Mobile & Web:** Cross-platform apps with **React Native (Expo)** and full-stack web with **Next.js**.
* **AI Integration:** Building vision-based ingredient recognition and LLM-backed recommendation engines (OpenAI GPT-4o).
* **Design Patterns:** Expertly adapting **CQRS, Mediator, and Event-Driven** approaches to manage complex business logic.
