### About

I design and build end-to-end software systems — from mobile apps to distributed backends. My focus is on clean architecture, maintainability, and integrating AI capabilities into real-world products.

Currently building a full-stack dietitian platform with a React Native mobile app, a Next.js web panel, and a .NET 8 backend featuring GPT-4o vision-based ingredient recognition and multi-tenant recipe recommendation.

---

### Core Competencies

- **Backend Engineering:** Enterprise-grade solutions with .NET 8 and Spring Boot ecosystems
- **Mobile Development:** Cross-platform apps with React Native and Expo (iOS & Android)
- **Web Development:** Full-stack with Next.js App Router, TypeScript, Tailwind CSS
- **Architecture:** Clean Architecture, Domain-Driven Design (DDD), strict layer separation
- **Distributed Systems:** Microservices, API Gateway integration, cross-service data consistency
- **AI Integration:** OpenAI GPT-4o vision pipelines, LLM-backed normalization, hallucination guards
- **Design Patterns:** CQRS, Mediator, Event-Driven, Repository, Saga (Choreography/Orchestration)
